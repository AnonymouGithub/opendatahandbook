Open Data Handbook

簡介¶

您可知道在您付出的稅金中，有多少是用來整修路燈或是有多少是使用在關於癌症的研究上？從您的住家到上班的地方，哪一條腳踏車專用道是距離最短，最安全，風景最迷人的？一路上的空氣品質又是如何？在您所居住的區域，哪裡可以找到最佳的工作機會？依照市民密度比來看，平均上哪邊又有著最多的果樹？對於您深切關注的政策主題，在什麼時候您能發表意見並產生影響？您又該對誰提出您的看法？

新科技現在可以建立能自動回答這些問題的服務。大多數您所需要用來回答這些問題的資料都是由公部門單位所產出。但是，這些資料常常處於一種非常難使用的狀態。這本手冊要講的就是如何釋放政府與非政府資訊所蘊藏的能量來開創新的服務，改善人民的生活和讓政府與整個社會運作的更完善。

關於 open data (資料開放) 和尤其是 open government data (政府資料開放) - 包含資訊，公開或是非公開，任何人都可自由取得與不受限制重複使用 - 這種概念已經被提出來有數年之久。在 2009 年 Open Data 開始出現在主流媒體上，同時一些政府（例如美國，英國，加拿大和紐西蘭）也宣布啟動開放他們的公共資訊的計畫。

這份手冊將會介紹關於 `open data’ 的基本概念，尤其是關於政府資訊的部分。本手冊的內容將說明 open data 如何製造加值與對許多不同領域可帶來的正面影響。除了這些基本的說明，本手冊也將提供關於如何實踐 Open Data 的實際的資訊。

預設讀者群¶

本手冊適合非常廣泛的讀者群：

  • 從那些從未聽過 Open Data 的人到那些自認為屬於季節性的 ‘資料專家’
  • 從國家公僕到社運人士
  • 從新聞記者到研究人員
  • 從政治人物到軟體開發者
  • 從資料技客 (data geeks) 到那些從未聽過 API 這個名詞的人

絕大部分提供的資訊在目前都是聚焦在以公部門所擁有的資料為主。但是，本手冊的作者們期望在時間允許下將更多的內容包含進來。我們歡迎大家一起來參與和幫助我們達到這個目的。

這本手冊主要是設計給那些對於這個主題不太熟習或是完全不知道的人。如果您發現某些特殊用語或是專業用字您不是非常清楚他們的含意，請參閱手冊最後的名詞說明 (Glossary) 與 常見問答集 (FAQs)。
