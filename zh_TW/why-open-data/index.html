為何要開放資料？¶

資料開放，尤其是政府資料的開放，代表的是一個幾乎完全沒有妥善開發使用的龐大資源。許多人和企業組織為了能完成他們的工作必需要收集範圍非常廣泛的各種資料。在這些資料收集者之中，如果從收集資料的量與類別來看，政府是其中最重要的一份子。但還有一個很重要的因素，政府所收集的資料，絕大部分在法令規定上屬於公共資料，所以可以開放釋出並提共他人使用。為什麼這一點很重要（有用）？

在很多不同的領域上我們都可期待資料開放本身就能帶來新的價值，在這一點我們已經看到許多實際的例子。同時對於許多群體與組織來言，他們也都能從資料的開放受益，這也包含了政府本身。同時，我們實在很難預測資料的價值會在何時以什麼樣的型態被創造出來。創新與新發明的本質就在於他們都出自於我們無法想像的地方。

It is already possible to point to a large number of areas where open government data is creating value. Some of these areas include:

  • 透明度與民主監督
  • 共同參與
  • 自我啟發
  • 改善或是創建新的商品與服務
  • 創新
  • 改善政府服務的效能
  • 改善政府服務的效力
  • 政策影響的度量
  • 經由分析與結合，從大量的資料中將資料來源與模式，產生的新知識

在這些領域中我們已經有成功的範例。

在透明度這一部分我們看到例如芬蘭的 `tax tree’ 和英國的 `Where does my money go’。這些網站將政府如何使用人民的稅金整理並顯示出來。或是來自加拿大的例子，看 Open Data 如何為加拿大政府從慈善稅金詐騙中省下 32 億美金。還有其他許多網站，例如丹麥的 folketsting.dk 追蹤國會動態與政策制訂的程序，所以人民可以清楚知道目前發生了什麼事，和有哪些國會議員參與。

政府資料的開放也可讓你在生活中做出更好的決定，或是讓你有更多的機會在社會上更活躍。一位丹麥的女士建立了一個叫做 findtoilet.dk 的網站，在這個網站上可以找到所有丹麥的公共廁所。這個網站讓許多原本因為膀胱問題不敢出門的人有更多自信和願意多多出門。在荷蘭，透過一個叫做 vervuilingsalarm.nl 的網站，人們可以在住家周遭空氣品質預低達到一個可自訂的標準時收到警告。在紐約市，您可以很容易的找到可以溜狗的地方，和找到其他也是使用這些公園的人們。在英國的 `mapumental’ 與在德國的 `mapnificent’ 的服務則可方便您依照不同的標準來找到適合您的住宅選擇，例如從住家到工作地點的通勤時間，房屋的售價，甚至依照居住環境的風景等。所有這些服務都是使用政府開放的資料。

從經濟上來說，open data 也是一樣非常重要。許多研究都估計，光是在歐洲共同市場，Open Data 能帶來的經濟價值就可達到每年數百億歐元。新的產品與新企業正在使用開放資料。丹麥的 husetsweb.dk 可幫助您找到改善居家能源在使用上的效率，包含財務規劃與找到施工承包商等。這個服務除了使用了地價資訊，還使用了政府補助資訊和地方商業登記資訊。Google 翻譯大量使用歐盟產出包含多國語言的文件來調校他們的翻譯演算法，進而改善服務品質。

對於政府本身來說，資料開放也有其價值。例如效能的改進。自從荷蘭教育部將他們所有與教育相關的資訊都公佈在網路上並開放使用後，除了因為所收到的問題數量減少而降低工作量與經費的需求。剩下來的問題也能輕易的由國家公僕來回答。因為這些問題的答案都可以更容易的被搜尋到。資料開放也讓政府更有效力，長期來看這也是降低政府支出成本的因素。荷蘭文化資產部積極的釋放出他們的資料並與業餘歷史學家社群與團體例如 Wikimedia Foundation，進行合作來更有效力的執行他們的工作。這一切不只帶來他們資料品質的改善，同時也讓整個部門能更精簡。

在許多不同的領域裡，雖然我們已經看到許多資料開放的運用所帶來，不管是在社會上或是經濟上的效應，我們依然不知道在未來還會有什麼創新。資料的重新組合可以創造出新的知識與對事物的理解，而這一切都可帶來在全新的領域上的運用。這在以前已經發生過了，例如，19 世紀時，透過因黑死病而死亡的病例與各地方水井的所在地建立連結，讓 Dr Snow 在倫敦發現了飲用水的污染和黑死病之間的關連。這個發現也讓倫敦興建了地下污水系統，並為市民的公眾衛生帶來極大的改善。經由開放資料產生的連結，我們也可期待對於這個社會與世界能有新的認知與發現。

如果我們將政府公共資訊轉為開放資料，這些未開發的潛能就能夠被進一步釋放出來。但是，只有當這一切真正開放，也就是沒有任何限制（法令上的，財務上的，或是技術上的），才能讓他人可以自由重新使用。任何一種限制都只會造成某一部分的人無法使用這些公共資料，並降低找到新的價值的可能性與方法。要讓這個潛能被釋放出來，公共資料必須成為開放資料。
